Variables

Variables hold data to be referenced later. They can hold all kinds of different data types. 


            // Number
            var x = 420;
            var y = 6.9;
            var z = -80085;
            // String
            var person = "Jimbo";
            var phrase = "Everyone's a critic";
            // Boolean
            var iAmCool = true;
            var tooCoolForSchool = false;
            // Array
            var people = ["Jimbo", "Sally", "Randy", "Carl"];
            var numbers = [2, 8, 4, 0, 0, 1];
            // Object
            var profile = {name: "John", age: 22, skills: ["juggling", "cooking", "archery"]};

            // reference a variable
            console.log("My name is " + Jimbo);

Arrays

Arrays are lists of values. Javascript has a few tools for working with arrays.


            var varietals = ["Chardonnay", "Merlot", "Pinot Noir", "Zinfandel"];

            // Get an item in an array by it's index
            console.log(varietals[2]); // Pinot Noir

            // add another item to the array
            varietals.push("Sauvignon Blanc");

Objects

Objects are a robust way to handle more complex data. Javascript has a bunch of tools to work with objects.


            //This is an object, it goes in curly braces
            // and consists of key/value pairs.
            
            var person = {
                name: "Timmy", // "name" is the key and "Timmy" is the value
                age: 100,
                hairColor: "green",
                "height (cm)": 100 
            }

This is the common way to reference the data


            console.log("Their name is " + person.name); // Their name is Timmy
                         // the variable____|      |
                       //   the key________________|

if the key contains spaces or special characters (-, (), etc.), then you have to put it in brackets and quotations instead.


            console.log("They are " + person["height (cm)"] + " centimeters tall");

Functions

Functions are reusable blocks of code that run when called.

Declaring a function

There are a few ways to declare a function, but this is the most common.


            function logToConsole(){
                console.log("this is a function");
            }

Calling a function

Once you have declared a function, you can make it execute like this:


            logToConsole();
            
            // Output:
            // this is a function

Parameters and Arguments

a parameter is like a special little variable just for the function. This function is going to get a parameter names "message".


            function logToConsole(message){
                console.log(message);
            }

When the function is called, we can set the value of the parameter. That's called an argument.


            logToConsole("I am passing this to the function");
            
            //Output:
            // I am passing this to the function

If / else

"if" and "else" are conditional statements. They are used to create logic.


            var today = "Friday";
            
            if (today === "Monday"){ // false, so it wont run this code
                console.log("This day sucks");
                }else if(today === "Friday"{ // true, so it will run this code
                    console.log("This day rules");
                }else{ // will run if nothing else did
                    console.log("I wish it was Friday");
                }

Logical operators

You can use "and" (&&), "or" (||) and "not" (!) to make more complex conditional statements


                    if (!dogsBarking && mommaCooked && gotACall && !killedInSouthCentral){ 
                        // will only run if all conditions are true
                        console.log("it was a good day");
                    }else if (lakers > superSonics || sheHadTheChronic){
                        // will run if either conditions are true
                        console.log("I got to say it was a good day");
                    }

Loops

Loops provide a way repeating code. Javascript offers a lot of different types of loops. These are the ones we have learned so far.

for loop

this loop repeats until a specified condition is met.


            for (var i = 0; i < 5; i++){
                console.log("This will be logged 5 times");
            }
            // (var i = 0) i is set to 0 
            // (i < 5) the loop is told to continue as long as i is less than 5 
            // (i++) the loop is told to increment the value of i each time it runs

forEach loop

This loop runs for each item in an array while passing the value (and, optionally, the index) of the array item.


            var ingredients = ["mayo", "mustard", "pickles", "ham"];
            
            ingredients.forEach(function(ingredient, i){
                console.log(i + ".) put the " + ingredient + " on the bread");
            });
            
            // Output:
            // 0.) put the mayo on the bread
            // 1.) put the mustard on the bread
            // 2.) put the pickles on the bread
            // 3.) put the ham on the bread

Events

Javascript can capture events like key presses and clicks.

                // and referenced here____________|

onclick

This is triggered whin a user clicks


            document.onclick = function() {
                console.log("You clicked");

Scope

Scoped Variables

Variables that are declared inside functions are only available inside that function.


        function makeVariable(){
            var x = 2;
        }
        
        makeVariable();

        console.log(x); // Can't do it.

Global variables

Global variables are defined outside of any functions, making them available... globally. They can still be changed inside a function, and the new value will be accessible anywhere.


            var x = 2;
            
            function changeVariable(){
                x = 10;
            }
            
            changeVariable();
            
            console.log(x); // 10

DOM Manipulation

This means changing things in the web page with JavaScript. There are a lot of ways to do it.

Selecting elements

If you want to change an element, you first need to select it. There are a bunch of ways to do that as well.


            // Select by id
            var story1 = document.getElementById("story-1");
            // This will select that element and store is in the variable

            // Select all elements with the class "story"
            var stories = document.getElementsByClassName("story");
            // This will return a list of matching elements (kind of like an array)

            // querySelector can select by just about anything
            var story1 = document.querySelector("#story-1"); 
            // selects the first one it finds

            // querySelectorAll returns a list of matches
            var stories = document.querySelectorAll(".story"); // all of that class
            var article = document.querySelectorAll("article"); // all article elements

Changing elements

Javascript can change just about anything about an element.


                // change the text in an element
                story1.textContent = "This is the new text"

                // put html right in an element
                story1.innerHTML = "<h1>this is a new h1 tag</h1>"

                // add a class to an element
                story1.classList.add("new-class");

                // add a style to an element
                story1.style.background = "#f06d06";
